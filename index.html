Mobile Claw 개인정보처리방침

시행일: 2026년 3월 23일

Mobile Claw는 사용자 기기에서 동작하는 Android 앱입니다. 개발자가 별도로 운영하는 자체 서버에 사용자 정보를 저장하지 않는 것을 원칙으로 합니다. 다만 앱 기능 제공을 위해 일부 정보가 제3자 서비스로 직접 전송될 수 있습니다.

1. 처리하는 정보

  • 로그인 시 필요한 OAuth 정보 및 인증 토큰
  • 사용자가 입력한 프롬프트, 첨부 이미지, 스크린샷
  • 앱 설정, 로컬 채팅 기록, 예약 작업 등 기기 내 저장 데이터
  • 기능 사용 시 필요한 권한 정보: 위치, 연락처, 캘린더, 마이크, 알림, 오버레이, 접근성
  • Firebase를 통한 앱 사용 정보 및 푸시 알림 관련 정보

2. 이용 목적

  • LLM 호출, OAuth 로그인, 작업 수행 및 결과 표시
  • 사용자 요청에 따른 위치 조회, 연락처 검색/생성, 캘린더 작업, 음성 입력 등 기능 제공
  • 오버레이 및 접근성 기능을 통한 앱 간 제어 지원
  • 푸시 알림, 앱 안정성 확인, 기본 통계 분석

3. 제3자 제공 및 외부 전송

앱은 다음 서비스로 정보를 직접 전송할 수 있습니다.

  • OpenAI: OAuth 로그인, 토큰 갱신, LLM 응답 처리
  • Google Firebase: Analytics, Cloud Messaging

이 과정에서 데이터는 국외 서버에서 처리될 수 있으며, 각 서비스의 처리방침은 해당 제공자의 정책을 따릅니다.

4. 기기 내 저장

인증 정보, 앱 설정, 일부 대화/작업 데이터는 사용자의 기기에 저장될 수 있습니다. 사용자는 앱의 로그아웃 또는 로컬 데이터 삭제 기능을 통해 이를 제거할 수 있습니다.

5. 권한 사용

앱은 기능 실행에 필요한 경우에만 권한을 요청합니다. 권한을 허용하지 않으면 해당 기능은 동작하지 않을 수 있습니다.

6. 문의

개인정보처리방침 관련 문의는 Google Play 스토어에 기재된 개발자 연락처를 통해 할 수 있습니다.

7. 변경

본 방침은 앱 기능 변경 또는 법령 변경에 따라 수정될 수 있으며, 변경 시 이 페이지를 통해 공지합니다.
